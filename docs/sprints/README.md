# Sprint Roadmap — PITH

**Total:** 6 sprints (6 weeks to launch)
**Start:** Phase 4 complete. All commands stable. 66 tests passing. Audit completed 2026-04-08.

## Context

PITH has completed four development phases (scaffold through license/export/build). The engine is functional but not shippable. The 2026-04-08 audit identified 2 P0 and 6 P1 security findings, plus critical performance issues. This roadmap takes PITH from "works on the developer's machine" to "ships to paying customers."

## Sprints

| Sprint | Name | Goal | Duration | Status |
|--------|------|------|----------|--------|
| 0 | Hardening | Fix all P0/P1 audit findings, CI pipeline | 1 week | Not started |
| 1 | Performance | Async ingest pipeline, connection reuse, concurrency | 1 week | Not started |
| 2 | Polish | UX refinements, error handling, config safety | 1 week | Not started |
| 3 | Integration Testing | End-to-end tests, cross-platform builds, schema pack testing | 1 week | Not started |
| 4 | Pre-Launch | Lemon Squeezy integration, packaging, assets, legal | 1 week | Not started |
| 5 | Launch | Ship binaries, first customers, monitoring | 1 week | Not started |
